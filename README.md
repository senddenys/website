# Denys Kovalevskyi — QA Tester Portfolio

Personal portfolio site built with vibe coding for fun and job hunting.

**Live:** [senddenys.github.io/website](https://senddenys.github.io/website/)

## About

Static one-page portfolio for a QA engineer. Brutalist layout, yellow accent (`#f5d042`), bold Anton headings, and a hero section with interactive bug icons that scatter on hover.

## Sections

- **Hero** — intro and animated bugs
- **About** — short bio
- **Impact** — stats (bugs found, improvements, experience)
- **Skills** — testing, tools, process
- **Certificates** — carousel with 12 certs, including Master's diploma
- **Testimonials** — quotes from teammates
- **Contact** — email, Telegram, LinkedIn

## Tech

- HTML, CSS, vanilla JavaScript
- Google Fonts: Anton + DM Sans
- Google Analytics 4 (`G-GDS7YCX90D`)
- GitHub Pages deployment from `main`

## Run locally

```bash
python3 -m http.server 8080
```

Open [http://localhost:8080](http://localhost:8080).
