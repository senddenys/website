# Деплой на GitHub Pages

## Кроки

1. Відкрий репозиторій: **https://github.com/senddenys/website**
2. Перейди в **Settings** (вкладка вгорі).
3. У лівому меню вибери **Pages** (під "Code and automation").
4. У блоці **Build and deployment**:
   - **Source:** Deploy from a branch
   - **Branch:** `main` / **(root)**
5. Натисни **Save**.

Через 1–2 хвилини сайт буде доступний за адресою:

**https://senddenys.github.io/website/**

Після кожного `git push origin main` GitHub автоматично оновлює деплой.
